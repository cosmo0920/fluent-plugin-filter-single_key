## Installation

### RubyGems

```
$ gem install fluent-plugin-filter-single_key
```

### Bundler

Add following line to your Gemfile:

```ruby
gem "fluent-plugin-filter-single_key"
```

And then execute:

```
$ bundle
```

```
$ fluent-plugin-config-format filter filter-single_key
```
